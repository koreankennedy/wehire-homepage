# WeHire 와이어프레임 (Wireframe)

## 1. 개요

이 문서는 WeHire 홈페이지의 각 페이지별 레이아웃 구조를 정의합니다.

### 표기법
- `[ ]` : 버튼/클릭 가능 요소
- `─`, `│` : 구분선
- `┌┐└┘` : 컨테이너/카드 경계
